Country Info API Guide

This API provides information about countries, including general data, population history, and service status.

Endpoints

1. Country Information

Retrieves general information about a country, including name, capital, languages, borders, flag, population, and cities.

Request:

GET /countryinfo/v1/info/{two_letter_country_code}{?limit=10}

Parameters:

  • two_letter_country_code: The 2-letter ISO 3166-1 country code (e.g., no for Norway).
  • limit (optional): Limits the number of cities returned, sorted in ascending alphabetical order.

Example: /countryinfo/v1/info/no

Example with limit: /countryinfo/v1/info/no?limit=10

2. Population History

Retrieves population data for a country, with an optional year range filter.

Request:

GET /countryinfo/v1/population/{two_letter_country_code}{?limit={startYear-endYear}}

Parameters:

  • two_letter_country_code: The 2-letter ISO 3166-1 country code.
  • limit (optional): Specifies a range of years (e.g., 2000-2020 to retrieve population data between those years).

Example: /countryinfo/v1/population/no

Example with year range: /countryinfo/v1/population/no?limit=2000-2020

3. Service Status

Checks the status of the API.

Request:

GET /countryinfo/v1/status

Example: /countryinfo/v1/status
